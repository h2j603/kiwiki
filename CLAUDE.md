# CLAUDE.md — Kiwiki (웹키위바이러스 위키)

> 클라우드/로컬 세션 공통 컨텍스트.

## 프로젝트

가상의 "웹키위바이러스" 종류를 정리한 위키. **Jekyll** 정적 사이트.

- `_config.yml` — Jekyll 설정
- `_layouts/`, `_includes/`, `_sass/` — 템플릿·스타일
- `content/`, `data/` — 위키 콘텐츠
- 바이러스 효과(글리치 등) 인터랙션이 있고, `wiki-title`은 효과에서 제외됨

로컬 실행: `bundle install && bundle exec jekyll serve`

## 전역 규칙 (모든 세션 공통)

- 대화·커밋 메시지·코드 주석 모두 **한국어**. 커밋은 명령형.
- 작업은 feature 브랜치 → `main` PR 생성 → 머지까지 완료.
- 이 프로젝트는 의도적 B급·바이러스 감성이 컨셉 — 디자인 시스템 규칙(그림자
  금지 등)보다 **기존 톤 유지**가 우선. 기존 스타일 문법을 따를 것.
- 작동하는 코드를 "개선" 명목으로 구조 바꾸지 않기.
