## 개요 / Overview

검색엔진의 이미지 검색 결과가 망막에 영구적인 잔상을 남기는 감각형 바이러스. 정상적인 잔상은 몇 초 내에 사라지지만, 이 바이러스에 의한 잔상은 영원히 지속되며 새로운 잔상이 기존 잔상 위에 계속 쌓인다. 감염자의 시야는 검색한 모든 이미지가 중첩된 영구적 이중 노출 상태가 되며, 시간이 지날수록 잔상이 축적되어 실제 세계가 수천 개의 반투명 이미지 뒤에 가려지게 된다. 감염자는 현실과 잔상의 구분이 점차 불가능해지며, 시각적 정보 과부하로 인해 일상적 기능이 마비된다.
{: .ko}

A Sensory-type virus in which search engine image results leave permanent afterimages on the retina. Normal afterimages fade within seconds, but those caused by this virus persist forever, with new afterimages continuously layering on top of existing ones. The infected person's vision becomes a state of permanent double exposure with every searched image superimposed, and as afterimages accumulate over time, the real world becomes obscured behind thousands of semi-transparent images. The infected person gradually becomes unable to distinguish reality from afterimages, and daily function is paralyzed by visual information overload.
{: .en}

## 감염 방식 / Infection Method

감염은 검색엔진의 이미지 검색 기능을 사용할 때 시작된다. 검색엔진 이미지 캐시 서버에서 손상된 이미지 압축 알고리즘이 특정 이미지를 렌더링할 때, 해당 이미지에 미세한 시각적 "앵커" 패턴이 삽입된다. 이 앵커 패턴은 망막의 광수용체에 비정상적으로 강한 잔상 반응을 유발하며, 정상적인 잔상 소거 메커니즘을 차단한다. 이미지 검색 결과를 2초 이상 응시할 때마다 영구 잔상이 하나씩 형성되며, 하루 50장 이상의 검색 이미지를 열람하면 감염이 가시적 수준에 도달한다. 고해상도 이미지와 강렬한 색상의 이미지일수록 잔상이 더 선명하게 남는다.
{: .ko}

Infection begins when using search engine image search functions. When a corrupted image compression algorithm on the search engine's image cache server renders certain images, subtle visual "anchor" patterns are inserted into those images. These anchor patterns trigger abnormally strong afterimage responses in the retina's photoreceptors and block the normal afterimage erasure mechanism. Each time an image search result is gazed at for more than 2 seconds, one permanent afterimage forms, and viewing more than 50 search images per day causes infection to reach a visible level. Higher resolution images and images with intense colors leave sharper afterimages.
{: .en}

## 웹 상 증상 / Web Symptoms

- 이미지 검색 결과를 닫은 후에도 해당 이미지의 윤곽이 화면 위에 반투명하게 남아 보임
- 브라우저 화면에 이전에 검색한 이미지의 잔상이 겹쳐 텍스트 가독성 저하
- 이미지 검색 기록에 본인이 검색하지 않은 이미지가 나타나며, 해당 이미지가 잔상으로 남아 있는 느낌
- 화면 밝기를 최소로 낮춰도 이미지 잔상의 밝기는 변하지 않음
{: .ko}

- After closing image search results, the outlines of those images remain visible as semi-transparent overlays on the screen
- Afterimages of previously searched images layer over the browser screen, reducing text readability
- Images the user never searched appear in image search history, and those images feel like they persist as afterimages
- Image afterimage brightness remains unchanged even when screen brightness is set to minimum
{: .en}

## 발현 양상 / Manifestation

- 눈을 감아도 검색한 이미지의 잔상이 보이며, 수면 시에도 잔상이 꿈에 침투
- 실제 풍경 위에 검색 이미지가 반투명하게 겹쳐 보여, 얼굴 인식과 공간 판단에 심각한 장애 발생
- 잔상이 수백 개 이상 축적되면 시야 전체가 이미지의 콜라주 상태가 되어, 현실의 윤곽선만 간신히 파악 가능
- 말기에는 잔상이 현실보다 선명하게 보여, 감염자가 잔상 속 이미지를 실제 사물로 착각하고 반응하기 시작
{: .ko}

- Afterimages of searched images are visible even with eyes closed, and during sleep they infiltrate dreams
- Search images appear as semi-transparent overlays on real scenery, causing severe impairment in face recognition and spatial judgment
- Once hundreds of afterimages accumulate, the entire visual field becomes an image collage, with only the contours of reality barely perceptible
- In terminal stages, afterimages appear more vivid than reality, and the infected begin mistaking afterimage content for real objects and reacting accordingly
{: .en}

## 경계 무너짐 단계 / Boundary Collapse Stages

1. 검색엔진 이미지 열람 후 화면에서 잔상이 정상보다 오래 지속 (웹)
2. 모든 디지털 화면에서 이전 이미지의 잔상이 축적되기 시작 (웹 확산)
3. 오프라인 시야에서 검색 이미지의 잔상이 겹쳐 보이기 시작 (현실 침투)
4. 잔상이 현실보다 선명해져 시각적 현실 인식이 완전히 무너짐 (현실 정착)
{: .ko}

1. Afterimages persist longer than normal on screen after viewing search engine images (Web)
2. Afterimages from previous images begin accumulating across all digital screens (Web spread)
3. Afterimages of searched images begin overlaying offline vision (Reality breach)
4. Afterimages become more vivid than reality, completely collapsing visual reality perception (Reality anchor)
{: .en}

## 대응 방법 / Countermeasures

- 이미지 검색 사용을 최소화하고, 검색 결과 이미지를 2초 이상 응시하지 않을 것
- 브라우저에 이미지 검색 결과의 해상도를 자동 저하시키는 확장 프로그램 설치
- 감염 초기에 어두운 환경에서 장시간 눈을 휴식시키는 "시각 단식" 실시 — 완전한 암실에서 4시간 이상 체류하면 초기 잔상이 일부 소거될 수 있음
- 이미지 검색 대신 텍스트 기반 검색을 사용하고, 이미지가 필요한 경우 텍스트 설명을 통해 간접적으로 확인
{: .ko}

- Minimize image search usage and avoid gazing at search result images for more than 2 seconds
- Install browser extensions that automatically reduce the resolution of image search results
- In early infection stages, practice "visual fasting" by resting eyes in a dark environment for extended periods — staying in a completely dark room for 4+ hours may partially erase initial afterimages
- Use text-based search instead of image search, and when images are needed, verify indirectly through text descriptions
{: .en}

## 발견 기록 / Discovery Log

2025년 가을, 한 사진 편집자가 "사진 위에 다른 사진이 계속 겹쳐 보여서 작업이 불가능하다"고 호소하며 안과를 방문했으나, 모든 안과 검사에서 이상이 발견되지 않았다. 편집자는 업무 특성상 하루에 수백 장의 이미지를 검색엔진에서 검색하고 있었으며, 해당 검색엔진의 이미지 캐시 서버에서 손상된 압축 알고리즘이 사용된 것이 확인되었다. 이후 같은 검색엔진의 이미지 검색을 빈번하게 사용하는 디자이너, 연구자 등에게서 동일 증상이 보고되며 바이러스로 식별되었다. 최초 감염자는 "내 눈이 구글이 되어버렸다 — 한 번 본 이미지를 절대 잊지 못한다"고 증언했다.
{: .ko}

Discovered in autumn 2025 when a photo editor visited an ophthalmologist complaining "other photos keep overlaying on top of my photos and I can't work," yet all eye examinations found no abnormalities. Due to the nature of their work, the editor searched hundreds of images daily through a search engine, and a corrupted compression algorithm was confirmed on that search engine's image cache server. Subsequently, identical symptoms were reported among designers, researchers, and others who frequently used the same search engine's image search, leading to identification as a virus. The first infected individual testified: "My eyes have become Google — I can never forget an image I've seen."
{: .en}

## 관련 바이러스 / Related Viruses

- [고스트핑 (GhostPing)](ghostping.md): 검색엔진 벡터를 공유하며, 고스트핑의 팬텀 알림이 애프터이미지의 잔상과 시각적으로 중첩되는 사례가 보고되었다. 고스트핑의 확인 강박이 이미지 검색 빈도를 증가시켜 애프터이미지 감염을 가속화할 수 있다.
- [크로마드리프트 (ChromaDrift)](chromadrift.md): 크로마드리프트에 의해 색상이 역전된 상태에서 애프터이미지가 형성되면, 역전된 색상의 잔상이 영구적으로 축적되어 이중 왜곡이 발생한다.
- [픽셀롯 (PixelRot)](pixelrot.md): 픽셀롯이 시각적 형태를 부식시키는 동안 애프터이미지가 잔상을 축적하면, 부식된 이미지의 잔상이 남아 시야가 "디지털 폐허"처럼 변한다.
{: .ko}

- [GhostPing](ghostping.md): Shares the search engine vector; cases have been reported of GhostPing's phantom notifications visually overlapping with AfterImage's persistent afterimages. GhostPing's verification compulsion can increase image search frequency and accelerate AfterImage infection.
- [ChromaDrift](chromadrift.md): When afterimages form while color is inverted by ChromaDrift, afterimages in inverted colors accumulate permanently, creating double distortion.
- [PixelRot](pixelrot.md): When AfterImage accumulates afterimages while PixelRot corrodes visual forms, afterimages of corroded images persist and the visual field becomes like "digital ruins."
{: .en}

## 참고 / Notes

- 애프터이미지 감염자가 축적한 잔상의 수는 일종의 "시각적 기록"으로서, 감염자의 검색 이력을 역추적하는 데 활용될 수 있다. 일부 연구자들은 이를 "망막 고고학"이라 부른다.
- 선천적으로 잔상 인지가 약한 사람(약 인구의 2%)은 애프터이미지에 대한 자연 내성을 가진 것으로 관찰되었으며, 이들의 시각 처리 메커니즘이 치료법 개발의 단서가 될 수 있다.
{: .ko}

- The number of afterimages accumulated by an AfterImage-infected person serves as a kind of "visual record" that can be used to reverse-trace the infected person's search history. Some researchers call this "retinal archaeology."
- People with inherently weak afterimage perception (approximately 2% of the population) have been observed to possess natural resistance to AfterImage, and their visual processing mechanisms may hold clues for developing treatments.
{: .en}
