# 미러링크 (MirrorLink)

## 개요

SNS에서 공유되는 특정 이미지 링크를 클릭하면 감염되는 감각형 바이러스. 감염자는 거울이나 반사면을 볼 때 자신의 모습이 미세하게 다르게 보이기 시작한다.

## 분류

| 항목 | 내용 |
|------|------|
| 유형 | 감각형 |
| 전파 경로 | SNS |
| 위험도 | 높음 |

## 웹 상 특성

### 감염 방식

SNS 타임라인에 "이 사진 속 이상한 점 찾음?"과 같은 문구와 함께 공유되는 이미지 링크를 통해 전파된다. 링크를 클릭하면 거울처럼 반사되는 이미지가 로딩되는데, 이 이미지를 3초 이상 응시하면 감염이 시작된다.

### 웹 상 증상

- 감염 후 SNS 프로필 사진이 좌우 반전되어 보임
- 셀카를 촬영하면 미세하게 비대칭인 결과물이 저장됨
- 화상통화 시 자신의 화면만 약간 딜레이가 걸린 것처럼 보임

## 현실 영향

### 발현 양상

- 거울을 볼 때 자신의 반사 이미지가 0.5초 정도 늦게 따라오는 느낌
- 반사면에서 자신의 표정이 실제와 미묘하게 다르게 보임
- 진행되면 거울 속 자신이 시선을 먼저 돌리는 것처럼 느껴짐

### 경계 무너짐 단계

1. SNS에서 이미지 열람 (웹)
2. 화면 속 자기 모습에 위화감 (웹-현실 경계)
3. 실제 거울에서도 위화감 발생 (현실 침투)
4. 반사면을 의식적으로 회피하게 됨 (현실 정착)

## 대응 방법

- 의심스러운 "이상한 점 찾기" 류의 이미지 링크 클릭 자제
- 감염 초기에 48시간 동안 모든 반사면을 가리고 디지털 화면 사용을 중단하면 자연 치유 보고 있음
- 화상통화 앱의 셀프뷰 기능을 비활성화할 것

## 발견 기록

2024년 가을, 특정 SNS 커뮤니티에서 "거울이 이상하다"는 게시글이 동시다발적으로 올라오면서 처음 인지됨. 공통적으로 같은 이미지 스레드를 열람한 사용자들이었으며, 원본 이미지의 출처는 아직 추적되지 않았다.
