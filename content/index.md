# Kiwiki - 웹키위바이러스 위키

웹을 통해 퍼지며, 현실과 가상의 경계를 무너뜨려 현실에도 영향을 미치는 바이러스, 웹키위바이러스(Web Kiwi Virus)의 종류를 정리한 위키입니다.

## 바이러스 목록

| 문서번호 | 이름 | 유형 | 전파 경로 | 위험도 |
|----------|------|------|-----------|--------|
| `WKV-SN-001` | [[viruses/mirrorlink|미러링크]] | 감각형 | SNS | 높음 |
| `WKV-CG-001` | [[viruses/echoloop|에코루프]] | 인지형 | 검색엔진 | 보통 |
| `WKV-EV-001` | [[viruses/glitchmoss|글리치모스]] | 환경형 | 직접접근 | 치명적 |

## 문서번호 체계

`WKV-[유형]-[일련번호]`

| 코드 | 유형 |
|------|------|
| SN | 감각형 (Sensory) |
| CG | 인지형 (Cognitive) |
| EV | 환경형 (Environment) |
| LG | 로직형 (Logic) |
