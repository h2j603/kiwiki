# 웹키위바이러스 분류 체계 / Web Kiwi Virus Taxonomy

[웹키위바이러스](../glossary/index.md#웹키위바이러스--web-kiwi-virus)는 웹을 통해 퍼지며, 현실과 가상의 [경계를 무너뜨리는](../glossary/index.md#경계-무너짐--boundary-collapse) 바이러스입니다.

[Web Kiwi Viruses](../glossary/index.md#웹키위바이러스--web-kiwi-virus) are viruses that spread through the web and [collapse the boundary](../glossary/index.md#경계-무너짐--boundary-collapse) between the virtual and the real.

## 유형별 분류 / Classification by Type

현실에 영향을 미치는 방식에 따라 분류합니다. / Classified by their method of affecting reality.

| 유형 / Type | 설명 / Description | 해당 바이러스 / Known Viruses |
|------|------|------|
| 로직형 / Logic | 사고방식, 판단력에 영향을 주는 유형 / Affects reasoning and judgment | [널위스퍼](../viruses/nullwhisper.md), [신택스레이스](../viruses/syntaxwraith.md), [팬텀탭](../viruses/phantomtab.md), [바이트셰이드](../viruses/byteshade.md) |
| 감각형 / Sensory | 시각, 청각, 촉각 등 감각에 영향을 주는 유형 / Affects senses (sight, hearing, touch, etc.) | [미러링크](../viruses/mirrorlink.md), [픽셀롯](../viruses/pixelrot.md), [하울링크](../viruses/howllink.md), [블라인드스팟](../viruses/blindspot.md) |
| 인지형 / Cognitive | 기억, 인식, 지각에 영향을 주는 유형 / Affects memory, perception, and awareness | [에코루프](../viruses/echoloop.md), [데드스크롤](../viruses/deadscroll.md), [포겟미낫](../viruses/forgetmenot.md), [루프홀](../viruses/loophole.md) |
| 환경형 / Environmental | 물리적 환경(기기, 공간 등)에 영향을 주는 유형 / Affects the physical environment (devices, spaces, etc.) | [글리치모스](../viruses/glitchmoss.md), [캐시베인](../viruses/cachevein.md), [스태틱블룸](../viruses/staticbloom.md), [보이드메일](../viruses/voidmail.md) |

## 전파 경로별 분류 / Classification by Vector

| 전파 경로 / Vector | 설명 / Description | 해당 바이러스 / Known Viruses |
|----------|------|------|
| SNS / Social Media | 소셜 네트워크를 통해 전파 / Spreads through social networks | [미러링크](../viruses/mirrorlink.md), [데드스크롤](../viruses/deadscroll.md), [팬텀탭](../viruses/phantomtab.md), [스태틱블룸](../viruses/staticbloom.md) |
| 메신저 / Messenger | 메시지/채팅 앱을 통해 전파 / Spreads through messaging/chat apps | [널위스퍼](../viruses/nullwhisper.md), [캐시베인](../viruses/cachevein.md), [하울링크](../viruses/howllink.md), [루프홀](../viruses/loophole.md) |
| 검색엔진 / Search Engine | 검색 결과를 통해 전파 / Spreads through search results | [에코루프](../viruses/echoloop.md), [블라인드스팟](../viruses/blindspot.md) |
| 이메일 / Email | 이메일을 통해 전파 / Spreads through email | [픽셀롯](../viruses/pixelrot.md), [바이트셰이드](../viruses/byteshade.md), [포겟미낫](../viruses/forgetmenot.md), [보이드메일](../viruses/voidmail.md) |
| 직접접근 / Direct Access | URL 직접 접속을 통해 감염 / Infection via direct URL access | [글리치모스](../viruses/glitchmoss.md), [신택스레이스](../viruses/syntaxwraith.md) |

## 위험도 / Severity Levels

| 등급 / Level | 설명 / Description | 해당 바이러스 / Known Viruses |
|------|------|------|
| 낮음 / Low | 웹 상에서만 증상이 나타남. 현실 영향 미미함 / Symptoms only appear on the web. Minimal real-world impact | *(현재 해당 없음 / None currently)* |
| 보통 / Moderate | 현실에 경미한 영향을 줄 수 있음 / May cause mild real-world effects | [에코루프](../viruses/echoloop.md), [픽셀롯](../viruses/pixelrot.md), [신택스레이스](../viruses/syntaxwraith.md), [팬텀탭](../viruses/phantomtab.md), [루프홀](../viruses/loophole.md) |
| 높음 / High | 현실에 뚜렷한 영향을 미침 / Causes distinct real-world effects | [미러링크](../viruses/mirrorlink.md), [널위스퍼](../viruses/nullwhisper.md), [데드스크롤](../viruses/deadscroll.md), [바이트셰이드](../viruses/byteshade.md), [하울링크](../viruses/howllink.md), [스태틱블룸](../viruses/staticbloom.md) |
| 치명적 / Critical | 현실에 돌이킬 수 없는 수준의 영향을 미침 / Causes irreversible real-world effects | [글리치모스](../viruses/glitchmoss.md), [캐시베인](../viruses/cachevein.md), [블라인드스팟](../viruses/blindspot.md), [포겟미낫](../viruses/forgetmenot.md), [보이드메일](../viruses/voidmail.md) |
